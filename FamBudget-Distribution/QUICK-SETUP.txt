# Quick Setup Guide

## 1. Install
- Right-click "FamBudget-Portable-Installer.bat"
- Select "Run as administrator"
- Choose "Quick Install"
- Wait for completion

## 2. Start
- Double-click "FamBudget-Launcher.bat"
- Or use desktop shortcuts

## 3. Mobile
- Install Expo Go on your phone
- Scan QR code from mobile server
